# JerSix - Soccer Jersey E-commerce

JerSix is a modern e-commerce platform specializing in soccer jerseys, offering a wide selection of authentic team jerseys from around the world.

## Features

- Browse and purchase soccer jerseys from various teams
- Shopping cart functionality
- Secure checkout process with MercadoPago integration
- Newsletter subscription
- Admin dashboard for product and order management
- Mystery box feature for surprise purchases

## Technologies Used

- HTML5, CSS3, JavaScript
- PHP for backend processing
- MySQL database
- MercadoPago payment integration
- PHPMailer for email notifications

## Setup Instructions

1. Clone this repository to your local environment
2. Set up a web server (e.g., XAMPP, WAMP) and configure PHP
3. Import the database files from the `database` folder
4. Install dependencies using Composer:
   ```
   composer install
   ```
5. Configure your environment variables for:
   - Database connection
   - MercadoPago credentials
   - SMTP settings for email

## Project Structure

- `admin/` - Administrative dashboard and management
- `Css/` - Stylesheet files
- `Js/` - JavaScript files
- `database/` - SQL files for database setup
- `img/` - Image assets
- `Productos-equipos/` - Product pages for different teams
- `vendor/` - Composer dependencies
